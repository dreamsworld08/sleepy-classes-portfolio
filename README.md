# Shekhar Dutt — Portfolio Website

A premium portfolio website for Shekhar Dutt, Co-Founder & Faculty at Sleepy Classes IAS.

## Overview

This is a single-page portfolio built with modern web technologies:
- **index.html** — Public-facing portfolio website
- **admin.html** — Private CMS dashboard for managing content

## Features

✨ **Sections**
- Hero with status indicator
- About with credentials
- Achievements (bento grid)
- Books & eBooks
- Media mentions
- YouTube playlist
- Twitter feed
- Consultation booking (Calendly)
- Collaboration links (Instagram, LinkedIn)

🎨 **Design**
- Light mode with warm off-white background (#FAFAF8)
- Ali Abdaal-inspired aesthetic
- Responsive design
- Smooth animations & scroll effects
- Glassmorphism accents

🔒 **Admin Dashboard**
- Protected login (default: admin / shekhar2026)
- Edit all site content in real-time
- Profile photo upload with preview
- Create custom pages
- Data persists via localStorage

## How to Use

### Public Site
Open `index.html` in a browser to view the portfolio.

### Admin Dashboard
1. Open `admin.html`
2. Login with credentials: `admin` / `shekhar2026`
3. Edit any section (Hero, About, Books, etc.)
4. Upload profile photo under About section
5. Changes persist automatically
